## Installation

Via Composer

```bash
composer require alexgithub987/ugyfel
```

## Usage

```bash
 $data_array = [
   "nev" => "Dezső Miskolc", 
   "email" => "mjeszt@dezsomiskolc.hu", 
   "telefon" => "06301234567", 
   "adoszam" => "asdasfaf", 
   "ceg" => "", 
   "cim_type" => "SZEKHELY", 
   "orszag" => "HU", 
   "irsz" => "6800", 
   "varos" => "Hódmezővásárhely", 
   "kozterulet" => "", 
   "kozterulet_jelleg" => "", 
   "hsz" => "", 
   "egyszeru_cim" => "6800 Hódmezővásárhely Makói ország út 77178 hrsz." 
]; 

$ugyfel = new Ugyfel;
$ugyfel->index($data_array);
```
